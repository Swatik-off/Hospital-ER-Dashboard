# 🏥 Hospital Emergency Room Dashboard (Excel Project)

## 📌 Project Overview

This project is an **interactive Excel dashboard** built using **Hospital Emergency Room data** to analyze patient flow, waiting time, admission trends, and demographic patterns.

The dashboard transforms **raw hospital records (9,216+ rows)** into **decision-ready KPIs and visual insights** for faster operational understanding.

## 🎯 Objective

The main objective of this project was to:

* **Track total patient visits**
* Monitor **average waiting time**
* Analyze **admission vs non-admission trends**
* Understand **patient demographics**
* Identify **department-wise load distribution**
* Build an **interactive reporting dashboard** for hospital performance monitoring

## 🔍 What I Did in This Project

### 📁 1) Data Cleaning

* Checked for **missing Patient IDs**
* Validated **duplicate records**
* Standardized date and categorical fields
* Verified raw dataset row count (**9,216 rows**)

### 📈 2) Data Analysis

* Built multiple **pivot tables** for patient trends
* Created **time-based analysis** for ER traffic
* Analyzed **peak patient load periods**
* Compared **admitted vs discharged cases**

### 🎨 3) Dashboard Development

* Designed a **single-page executive dashboard**
* Added **interactive slicers** for drill-down analysis
* Used **dynamic KPI cards**
* Connected charts with filters for quick decision-making

### 🧠 4) Debugging & Validation
